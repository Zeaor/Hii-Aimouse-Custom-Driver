# Hii Aimouse 自定義驅動程式 (Python 版)

這是一個為 **Hii Aimouse** 語音滑鼠開發的開源驅動替代方案。由於原廠軟體已無法使用，本專案透過 Python 攔截 HID 訊號，實現按鍵自定義功能。

## ✨ 功能特色
* **完全自定義**：可設定麥克風鍵、搜尋鍵、側鍵的功能。
* **語音輸入**：支援「按住說話」模式 (Push-to-Talk)，使用 Google 語音辨識。
* **DPI 切換**：可透過側鍵快速切換滑鼠速度 (軟體加速)。
* **圖形化介面**：內建 GUI 設定工具，操作直覺。
* **多模式切換**：支援儲存多組設定檔 (Profile)，如文書模式、遊戲模式。
